# Rebranding & Feature Organization Analysis

## Current State: WebWatcher

### What It Actually Does

**Cybersecurity Functions:**
1. **Phishing Detection** - Scans websites for phishing red flags (A2A coordination)
2. **CVE Search** - Searches vulnerability databases
3. **Threat Intelligence** - Real-time security research via Exa MCP
4. **Log Analysis** - Analyzes security logs for incidents

**Web3 Functions:**
1. **Wallet Analysis** - Audits wallet addresses across multiple chains
2. **Transaction Analysis** - Examines blockchain transactions for risks
3. **ENS Resolution** - Resolves ENS domains to addresses
4. **Cross-Chain Monitoring** - Tracks cross-chain transactions (ZetaChain)
5. **USDC Tracking** - Monitors stablecoin flows (Circle)
6. **Project Vetting** - Verifies Web3 project legitimacy (Seedify)
7. **Network Monitoring** - Tracks blockchain performance (Somnia)
8. **Node Health** - Monitors validator infrastructure (NodeOps)

### Problem with "WebWatcher"
- Too generic - doesn't convey Web3 focus
- "Watcher" implies passive monitoring, but it's an active AI agent
- Doesn't highlight the AI/autonomous aspect
- Could be confused with general web monitoring tools
